Adding Files

TVRenamer provides several ways to add video files for renaming.

Drag and Drop

The easiest way: simply drag video files from Windows Explorer and drop them onto the TVRenamer window.

  • You can drag individual files or entire folders
  • Folders are scanned recursively for video files
  • Non-video files are ignored

File Menu

Use File → Add Files to open a standard file selection dialog.

Preload Folder (Automatic)

Configure a folder in Preferences that TVRenamer will automatically scan on startup:

  1. Open File → Preferences
  2. Find "Preload folder" and enable it
  3. Select or type the folder path
  4. Save preferences
Tip: Use the preload folder for your download directory. TVRenamer will automatically show new episodes when you launch it.

Supported File Types

TVRenamer recognizes these video extensions:

  • .avi, .mkv, .mp4, .m4v, .mov
  • .wmv, .webm, .flv, .ogm
  • .ts, .m2ts, .mpg, .mpeg
  • And more...

Filename Parsing

TVRenamer extracts show name and episode information from filenames. It recognizes patterns like:

Pattern Example
SxxExx Breaking.Bad.S01E01.mkv
sXXeXX breaking.bad.s01e01.mkv
XxXX Breaking Bad 1x01.mkv
Multi-episode Show.S01E01-E02.mkv or Show.S01E01-02.mkv
Series_X_-_YY Show_Series_3_-_18.Title.mkv

When Parsing Fails

If TVRenamer can't parse a filename, the "Proposed Name" column will show a specific reason:

  • No show name found - Can't identify the show name portion
  • No season/episode pattern - Missing S01E01-style numbering
  • Filename too short - Not enough characters to parse
Note: After adding files, TVRenamer shows a summary dialog if any files couldn't be parsed, listing the count and reasons.

Removing Files from the List

  • Select files and press Delete
  • Or right-click and choose "Remove"

This only removes them from TVRenamer's list, not from disk.
