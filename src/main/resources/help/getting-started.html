Getting Started

Requirements

  • Windows 64-bit (primary supported platform)
  • Java 17+ runtime
  • Internet connection (for TheTVDB lookups)

First Launch

  1. Download TVRenamer.exe or the JAR file
  2. Run the application
  3. On first run, you may want to open File → Preferences to configure:
    • Destination directory (where renamed files go)
    • Rename format (how filenames are structured)
    • Episode ordering preference (Aired vs DVD)

Quick Start: Rename Your First File

Step 1: Add Files

Drag and drop video files onto the TVRenamer window, or use File → Add Files.

Step 2: Wait for Lookup

TVRenamer will parse the filename, identify the show, and look up episode information from TheTVDB. You'll see:

  • Current Name - The original filename
  • Proposed Name - What it will be renamed to
  • Status - Whether the file is ready to rename

Step 3: Review and Fix Issues

If TVRenamer can't identify a show:

  • Check if the filename has recognizable show name and episode numbers
  • Use Show Overrides in Preferences to help matching
  • If multiple shows match, you'll be prompted to choose the correct one

Step 4: Rename

Select files and click Rename Selected, or press Ctrl+R.

Tip: Use Ctrl+A to select all files, then Ctrl+R to rename everything at once.

Understanding the Interface

Main Table Columns

Column Description
Current Name Original filename (before renaming)
Proposed Name New filename (after renaming). Shows error messages if parsing failed.
Status Icon showing ready (✓), error, or in-progress state

Keyboard Shortcuts

Shortcut Action
Ctrl+A Select all files
Ctrl+R Rename selected files
Ctrl+P Open Preferences
Ctrl+Q Quit application
Delete Remove selected files from list
