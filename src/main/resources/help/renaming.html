Renaming Files

Basic Renaming

  1. Add files to TVRenamer
  2. Wait for show/episode lookups to complete
  3. Review the "Proposed Name" column
  4. Select files to rename (or Ctrl+A for all)
  5. Click Rename Selected or press Ctrl+R

Rename Format

Customize how renamed files are named using format tokens. Open Preferences to edit the Rename Format.

Available Tokens

Token Description Example Output
%S Show name Breaking Bad
%s Season number 1
%0s Season number (zero-padded) 01
%e Episode number 1
%0e Episode number (zero-padded) 01
%t Episode title Pilot
%r Resolution (if detected) 720p

Example Formats

Format Result
%S S%0sE%0e %t Breaking Bad S01E01 Pilot.mkv
%S - %sx%0e - %t Breaking Bad - 1x01 - Pilot.mkv
%S/%S S%0sE%0e %t Breaking Bad/Breaking Bad S01E01 Pilot.mkv
Tip: Use forward slashes (/) in the format to create subdirectories. The show name can be used to organize files into folders.

Destination Directory

Configure where renamed files are placed:

  • Same folder - Rename in place (default if no destination set)
  • Custom destination - Move files to a specific folder (e.g., your media library)

Set this in Preferences → Destination Directory.

Multi-Episode Files

TVRenamer handles files containing multiple episodes:

  • Recognized patterns: S01E01-E02, S01E01-02, 1x01-02
  • Episode titles are concatenated with " & "
  • Example: Show S01E01-E02 Pilot & Day One.mkv

Conflict Handling

When a file with the same name already exists at the destination:

Default Behavior

TVRenamer creates a versioned copy: Show S01E01 Pilot (1).mkv

Always Overwrite Option

Enable "Always overwrite existing files" in Preferences to replace existing files instead of creating versions.

Warning: The overwrite option will permanently replace existing files. Use with caution.

Ambiguous Episode Titles

Some shows have different episode titles between aired and DVD orderings. When an episode number matches titles from both orderings, TVRenamer shows a dropdown with both options so you can choose the correct one.

Automatic Title Selection

TVRenamer can often pick the right title automatically:

  • Fuzzy matching from filename: If the filename contains the episode title (e.g., CHiPs.S03E18.Off.Road.1080p.mkv), TVRenamer extracts "Off Road" and matches it against the options to pre-select the best one.
  • Chain propagation: When you select a title for one episode, TVRenamer automatically cascades the choice through related episodes. For example, if episodes 18–20 share overlapping titles, selecting one determines the rest of the chain.

Automatic selections are non-destructive — the dropdown keeps both options and you can override any choice, which re-triggers propagation from that point.

Episode Ordering

TheTVDB provides multiple episode orderings:

  • Aired Order - Original broadcast order (default)
  • DVD Order - Order on DVD/BluRay releases (may differ for some shows)

Change this in Preferences → Episode Ordering.

Note: Some shows have different episode numbering between aired and DVD releases. If your files use DVD numbering, switch to DVD Order in Preferences.
