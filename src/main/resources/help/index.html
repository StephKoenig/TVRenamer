Quick Overview

TVRenamer takes ugly filenames like:

Lost.[6x05].DD51.720p.WEB-DL.AVC-FUSiON.mkv

And renames them to:

Lost S06E05 Lighthouse.mkv

Basic Workflow

  1. Add files - Drag and drop, use File menu, or set a preload folder
  2. Review matches - TVRenamer looks up show/episode info from TheTVDB
  3. Resolve any issues - Fix unrecognized shows or disambiguation prompts
  4. Rename - Click "Rename Selected" or use Ctrl+R

Key Features

  • Batch renaming of multiple shows at once
  • Customizable filename format with tokens
  • DVD/BluRay ordering support (not just aired order)
  • Multi-episode file handling (S01E01-E02)
  • Optional metadata tagging for MP4 and MKV files
  • Theme support (Light, Dark, Auto)
  • SMB/network share support

Getting Help

  • Report issues on GitHub
  • Enable debug logging with -Dtvrenamer.debug=true for troubleshooting
